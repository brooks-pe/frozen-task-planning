Expand KPI Number Hit Area + Add Hover Background

Update the clickable KPI numbers on the Task Planning Dashboard.

This is a targeted interaction refinement. Do not change the KPI layout, labels, rail colors, or navigation behavior.

1. Expand the clickable hit area for KPI numbers

The clickable target for each KPI number is currently too tight. Users have to hover directly over the numeral itself.

Increase the clickable / hoverable hit area so it extends slightly to the right of the number, creating a more forgiving interaction zone.

Requirements:

Keep the number as the visual focal point

Do not make the whole card clickable

Do not make the hit area so large that it feels like the card itself is interactive

The expanded hitbox should feel like a compact metric link area, not a button

2. Add a subtle hover background

When the user enters the KPI number hit area, show a subtle hover background behind the interactive region.

Requirements:

Use a very light background tint or soft neutral/accent wash

Keep it subtle and enterprise-appropriate

Rounded corners are acceptable if they align with existing design language

The hover background should help communicate:

this is interactive

the hit area is slightly larger than just the number

3. Preserve existing number styling

Keep:

accent blue number color

hoverable link behavior

current hierarchy and positioning

The number should still read as the primary metric, not a button label.

4. Preserve navigation behavior

Clicking the KPI number hit area should still:

navigate to the Tasks page

apply the corresponding Task Status Focus filter

Do not change the mapping.

5. Keep hover treatment lightweight

Do not add:

heavy fill colors

strong borders

shadows

button-like styling

animations beyond simple hover feedback if already used elsewhere

This should remain a metric interaction, not a CTA button.

6. Apply consistently across all four KPI cards

Use the same interaction treatment for:

Tasks Awaiting My Action

Stalled Tasks

Tasks Near Deadline

Overdue Tasks

7. Outcome goal

The KPI numbers should now feel:

easier to hover

easier to click

clearly interactive

without making the cards feel cluttered or over-designed.
