# Typography

- **Scope:** Font tokens, text roles, KPI metrics
- **Do not apply outside this domain**
- **Numeric alignment in tables follows Tables guidelines.**

## Rule Priority
- Hard Rules: must be followed exactly
- Guidance: preferred, but flexible

---

## Hard Rules

### Token Enforcement
- All text maps to a defined role. No ad-hoc sizes.
- Roles: Display Primary, Display Subtitle, Section Title, Container Title, Card Title, Body, Helper, Field Label, Field Value, Table Column Header, Breadcrumb.
- If a role doesn't exist, define it first. Do not improvise.
- Forbidden: 15px, 17px, inline font-size overrides, random semibold/medium mixing.

### KPI Metrics
- Numbers: Display Primary, semibold, default color.
- Labels: Card Title or Body.
- Metadata (timestamps): Helper.
